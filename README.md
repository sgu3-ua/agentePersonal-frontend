# agentePersonal-frontend
Este frontend únicamente esta preparado actualmente para LMStudio, de momento recomiendo el uso de la propia aplicación de LMStudio ya que contiene muchas más funcionalidades respecto al control del LLM. Únicamente permite la comunicación con el LLM requiere un token de LMStudio y el acceso a MCP.json

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```
